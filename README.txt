Sample Shopping cart using Angularjs
------------------------------------

A shopping cart application using angularjs.

how to run
----------

-install Xampp from below link
https://www.apachefriends.org/download.html
-put the extracted folder of project in to htdoc folder of xampp("Example"C:\xampp\htdocs)
-open xampp control panel and Run Apache server.
-Now run the project in any browser ("example": http://localhost/TestShopCartExample/Index.php.
